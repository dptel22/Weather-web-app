# Weather Forecast Web App

This project is a Streamlit-based web application that displays short-term weather forecasts for any city using live data from an external weather API.

## Overview
The app allows users to select a location, forecast duration, and data type to visualize upcoming weather conditions through interactive charts and images.

## Tech Stack
- Python
- Streamlit
- Plotly
- Requests
- OpenWeather API
- python-dotenv

## Features
- Search weather forecasts by city
- Select forecast range (1–5 days)
- Visualize temperature trends
- Display sky conditions using icons
- API error handling and validation

## Files
- `backend.py` – Handles API requests and data processing
- `main.py` – Streamlit frontend and visualization logic
- `project.py` – Additional data visualization example
- `requirements.txt` – Project dependencies

## Usage
1. Install dependencies
2. Set environment variable
  API_KEY=your_openweather_key
3. Run the app


## Learning Outcome
This project demonstrates API consumption, interactive data visualization, and building user-friendly web apps with Streamlit.
